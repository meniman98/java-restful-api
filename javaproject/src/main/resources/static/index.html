Welcome to Ahmed's web API hosted on Azure!

Go ahead and test my web API. Use a tool like postman or whatever you may prefer

To perform a GET request, type ("/cars/{id}") after the url to retrieve a single JSON object

To retrieve all JSON objects, simply type "/cars" at the end of the url

To perform a DELETE request, type ("/cars/{id}") and specify your ID

To perform a PUT request, type ("/cars/{id}") and specify your ID. Then fill in the object's new values. The keys are the following:

  • String brand
  • String type
  • String model

So for example, if we did a PUT request, it will look like this:

  • "brand": "BMW"
  • "type": "Coupe"
  • "model": "M4"

Now finally, to perform a POST request, type "/cars" after the url. No need to specify an ID. You will once again need to specify the values for the keys I mentioned above
